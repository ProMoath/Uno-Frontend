# Uno Website - Modern Frontend Interface

Uno Website is a comprehensive, multi-page web platform designed with a focus on high-quality User Interface (UI) and seamless User Experience (UX). This project showcases a modern approach to building service-oriented websites using core web technologies.

---

## 🌟 Key Features
* **Complete User Journey:** Includes everything from landing pages to authentication flows.
* **E-commerce Ready:** Features a dedicated store interface (`Moath_Store`) for product showcasing.
* **Interactive Components:** Utilizing JavaScript for dynamic elements and improved user engagement.
* **Responsive Design:** Styled with custom CSS to ensure a consistent look across various screen sizes.
* **Modular Structure:** Organized directory system for scalable development.

---

## 🛠️ Tech Stack
* **HTML5:** Semantic markup for structured and accessible content.
* **CSS3:** Custom styling featuring modern layouts and aesthetic designs.
* **JavaScript:** Client-side scripting for interactive features.

---

## 📂 Project Navigation
The repository is structured as follows:
* `index.html`: The main landing page.
* `Moath_Store.html`: Product catalog and shopping interface.
* `Services.html`: Detailed service offerings.
* `log-in.html` & `Sign up.html`: Full authentication UI.
* `Subscription.html`: Membership and pricing tiers.
* `Contact.html`: User inquiry and support interface.
* `/CSS & JS`: Core logic and styling assets.
* `/Images`: Optimized visual assets.
